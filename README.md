# ErronkaBAIETZ
BAIETZ Erronkako Repositorioa

Proiektu honetan, Bergarako loteria-administrazioak burututako eskaerari erantzuten zaio.
Loteria karpetan aplikazoaren kode guztia jaso da.
Bergarako loteria administrazioak, bere erabiltzaileek jokatutako apustuen emaitzak kalkulatzeko programa bat eskatu du.
Programa hau kontsola bidez exekutatu beharko da eta bertan “Primitiva”, “Euromillones” eta “La Quiniela”-ko apustuen emaitzak lortu beharko dira.
Erabiltzaile bakoitzak egin duen apustua sartu ondoren, programak burututako apustuak saria duen edo ez itzuliko du.
Programa hau Eclipse garapen ingurunean garatuko da, Git teknologia erabiliaz.
